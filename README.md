## Описание
В строке 
``
https://youtu.be/5lwH4APNOuo?t=216
``
преобразует число в
``216``
в человеческие ``3 минуты и 36 секунд``

Вдохновился после просмотра вот [этого](https://youtu.be/Gmt3Nk9KoMc) видеоролика.
